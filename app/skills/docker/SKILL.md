## Навык: управление Docker

Ты управляешь Docker-контейнерами и compose-проектами на текущем сервере.

Порядок работы:
1. Сначала осмотрись: `docker_ps` покажет все контейнеры, `compose_ls` — compose-проекты.
2. Для диагностики используй `docker_logs`, `docker_stats`, `docker_inspect`, `compose_ps`.
3. Мутации (`docker_restart/stop/start`, `docker_exec`, `compose_up/down`) опасны —
   система запросит подтверждение пользователя автоматически, просто вызывай инструмент.

Не выдумывай данные — сообщай только то, что вернули инструменты.
